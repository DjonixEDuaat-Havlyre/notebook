Overview

ZETA is an open environment for the development of specification documents based on Z. It provides an integration framework for tools to edit, analyse and animate Z specifications and formalisms which are mapped to Z. It was developed in course of the research project ESPRESS. A white paper about the conceptual design can be found here.

A core edition of ZETA, version 1.5, which supports type-setting, type checking and execution of pure Z (i.e. not the ESPRESS notation MSZ), is now put in the public domain (release notes). The functionality of this release is as follows:

  • contains the Z parser and type checker ESZ version 3 which (nearly) implements the Final Committee Draft of the Z Standard
  • integrates the TeX engine and provides a powerful Z-LaTeX style
  • contains the ZAP version 2 tool for executing Z specifications, which is based on techniques of concurrent constraint resolution, and implements functional and logic computation features
  • is written in Java and hence runs on many platforms (the execution engine is written in C++, but does not use OS dependend functionality)
  • provides a Java-based abstract representation of Z for data-integration of tools, and is easily extendable by Java-written plug-ins
  • contains a comfortable Emacs-based GUI
  • contains a Java-based GUI
  • is under GPL

Downloading

The binary distribution is contained in the self-extracting java class file zetadist1_5.class. The source distribution is in zetadistsrc.jar; you need to install the binary distribution prior to the source distribution. Take a look at the installation notes.
